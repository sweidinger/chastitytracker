Design-Tokens (aus globals.css)

lock-bg / lock-border
unlock-bg / border
inspect-bg / border
background / border

Design-Regeln

  • Hero bleibt: Gradient lock-bg → d1fae5, Dauer prominent mit "DAUER"-Label
  • Pills: Bestehende Komponente (h-7, rounded-full, border, bg-*-bg, text-*-text)
  • Session-Karte: weiß, border #e5e7eb, rounded-2xl, shadow-card
  • Milestone-Strip: dezent auf background-subtle, Lock-Farbe für Text (kein amber)
  • Inline-Marker: Lock-bg Streifen statt amber, ★-Symbol
  • Bucket-Header: 48px hoch, Hover: background-subtle, Count-Pill neutral
  • Condensed-Row: 32×32 Icon-Kreis mit semantic-bg + border
  • Historische Sessions: nur absolute Labels, START/ENDE-Events als Hero-Streifen
